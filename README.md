# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1ohNwR7dStMSKc8-bRZSA_b6-utleKmAE

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create a `.env.local` file by copying `.env.example`.
3. Set the `VITE_GEMINI_API_KEY` in your new `.env.local` file to your Gemini API key.
4. Run the app:
   `npm run dev`
